Format attendu

Le fichier Excel doit avoir la structure suivante :

  • Ligne 1 : En-tête du rapport (JOBSCHED / PD371)
  • Ligne 2 : Informations d'impression
  • Ligne 3 : En-têtes des colonnes : # produit | Description | Format | Provenance | Prix
  • Ligne 4+ : Données des produits
Format accepté : Liste prix Price list.xlsx
Les anciens prix seront automatiquement désactivés à cette date
Traitement automatique
  • Normalisation des noms
  • Détection des unités
  • Gestion des doublons
  • Mise à jour des prix
Exemple de données
06314 | Boîte 19x13x10 275C | 1UN CS | CAN | 1.75
06715 | Enregistreur de Température | 1UN CS | USA | 63.75
06415 | Étiquette Blanche | 3000UN CS | USA | 66.25
Important
  • Les ingrédients existants seront automatiquement reconnus (insensible à la casse)
  • Les nouveaux ingrédients seront créés avec l'unité détectée depuis le format
  • Les codes produits seront sauvegardés comme référence fournisseur
  • Le format (ex: "1000UN CS") sera sauvegardé comme conditionnement

Aide
Processus d'import :
  1. Téléchargez le fichier Excel depuis Hector Larivée
  2. Vérifiez que le format correspond (3 lignes d'en-tête)
  3. Sélectionnez le fichier et la date
  4. Cliquez sur "Importer"
Normalisation des noms :

Le système nettoie automatiquement :

  • Les dimensions (19x13x10)
  • Les codes produits (275C)
  • La casse (boite → Boîte)
  • Les accents manquants
Unités détectées :
CS → Caisse UN → Unité KG → Kilogramme BTE → Boîte
